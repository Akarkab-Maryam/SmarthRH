# SmartHR

SmartHR est une application de gestion des ressources humaines moderne conçue pour simplifier la gestion des employés, des présences, des congés et des documents au sein d'une organisation.

### Fonctionnalités :
* **Gestion des Utilisateurs :** Inscription, connexion sécurisée et gestion des profils.
* **Gestion des Employés :** Suivi complet des fiches employés, départements et postes.
* **Présences (Attendance) :** Pointage et suivi du temps de travail.
* **Congés (Leave Management) :** Demandes de congés et validation.
* **Gestion Documentaire :** Upload et stockage de documents (CV, contrats, etc.).
* **Tableau de bord :** Vue d'ensemble pour les employés et les administrateurs.
* **Notifications :** Système d'alertes pour les actions importantes.

### Technologies utilisées :
* **Backend :** Java 17+, Spring Boot 3
* **Sécurité :** Spring Security
* **Moteur de template :** Thymeleaf (HTML5/CSS3/JS)
* **Gestion de base de données :** Spring Data JPA (Hibernate)
* **Build Tool :** Maven
* **Conteneurisation :** Docker
